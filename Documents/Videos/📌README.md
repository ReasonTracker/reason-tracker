# 📌 Videos

This folder is the source of truth for video scripts, research, and most episode-specific assets.

## Conventions

- Episode folders use the format `Episode0001`, `Episode0002`, and so on.
- User-facing labels should remove zero padding.
- Remotion code belongs in `Documents/technical/Software/Video`.
- Shared reusable media belongs in the Video package; episode-specific media belongs in the relevant episode folder here.
